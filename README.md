# Customers-Insights-Dashboard
An interactive Power BI Dashboard built to analyze customer behavior, sales performance, campaign effectiveness, delivery operations, and regional business insights for an e-commerce business.
This project helps businesses track KPIs, monitor customer engagement, optimize marketing campaigns, and improve operational efficiency using data-driven insights.

Project Overview:
This dashboard provides a complete analytical view of:
Customer Insights & RFM Analysis
Regional Sales Performance
Marketing Campaign Performance
Channel-wise Revenue Analysis
Delivery & Return Analytics
Customer Segmentation
Revenue & Spend Tracking

Dashboard Pages
 Regional & Operational Insights
Key Analysis:
RFM score by region
Frequency of orders by location category
Return rate by area type
Net amount by region
Refund amount trends
Geo-location analysis using maps
Insights:
North and South regions generate the highest revenue
Semi-Urban areas show higher return rates
Tier-3 locations contribute the highest order frequency

Delivery Performance Dashboard
Key Analysis:
Completed deliveries by year
Returned deliveries by month
Late vs On-time deliveries
Spend vs Revenue trends
Restocking fee analysis
Insights:
On-time deliveries remain consistently higher
Revenue declined in 2026 compared to 2025
Return deliveries remain stable across months
